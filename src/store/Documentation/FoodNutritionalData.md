# List of APIs for Healthcare Related Services

## `Food Nutritional Data`

`# Search a food/recipe item (100g serving)`

Search a food item and get all its nutritional information

```
GET `https://food-nutrional-data.p.rapidapi.com/${nutrientQuery}`
```
